# Health Risk Detection System

This project is a machine learning-based system that predicts health risks based on user input data.

## Technologies Used
- Python
- Machine Learning
- Pandas, NumPy, Scikit-learn

## Features
- Predicts health risk level
- Data preprocessing and analysis
- Model training and evaluation

## How to Run
1. Install required libraries
2. Run the main Python file
